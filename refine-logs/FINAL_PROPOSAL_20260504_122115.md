# Final Proposal

**项目名**: DOPPLER - Dual-guided Offline RL for Traffic Sensor Layout Optimization  
**日期**: 2026-05-04  
**目标期刊**: Transportation Science  
**最终判定**: READY FOR PILOT

## Problem Anchor

在给定交通网络、历史全网状态数据和有限传感器预算的情况下，选择一组固定传感器节点，使部署后只观测这些节点的交通状态时，CDSTE 能尽可能准确地估计全网交通状态。

这个 anchor 排除三类漂移：

- 不把主任务改成重新设计 CDSTE 估计器。
- 不把主任务改成交通信号控制或移动传感器 relocation。
- 不把主任务改成纯 ML4CO benchmark；必须保留 OR formulation、可解释 baseline 和 certificate。

## Method Thesis

传感器布局是一个昂贵的组合优化问题：真实目标由 CDSTE 估计误差定义，评估代价高且不可微。DOPPLER 用 AMPL/MIP 松弛、对偶价格、可观测性 surrogate 和局部搜索轨迹构造结构化训练数据，再训练离线/序贯 RL policy 进行 add/swap sensor 决策，从而在有限 estimator evaluations 下获得高质量、可解释、跨预算可迁移的布局。

## Dominant Contribution

**OR-structured learning for traffic sensor layout**：不是把 RL 直接扔进选址，而是让 OR 模型提供 primal-dual guidance、candidate generation、budget shadow price、surrogate gap 和 layout stability evidence；RL 的作用是 amortize expensive search。

## Method Components

1. **Estimator objective**
   - 输入：sensor set `S`, budget `|S| <= B`, historical traffic matrix `X`, graph/spatial matrix `A`。
   - 输出：CDSTE 在 hidden nodes `V \ S` 上的 MAE/RMSE/MAPE/CRPS。
   - 约束：部署推理阶段只允许使用 `S` 的观测值。

2. **OR surrogate layer**
   - Coverage/facility-location MIP：覆盖交通相关性或空间邻近代表性。
   - Null-space / observability metric：借鉴 partial observability 文献，评估拓扑信息损失。
   - Correlation/covariance surrogate：最大化被观测节点对未观测节点的解释力。
   - LP/MIP relaxation：输出 reduced cost、dual price、budget shadow price、fractional solution。

3. **Search-data generation**
   - Greedy add trajectory。
   - Swap/local-search improvement trajectory。
   - AMPL solution pool / incumbent sequence。
   - BO-lite candidate sequence 作为强 baseline。

4. **RL policy**
   - 状态：当前 sensor set、预算剩余、节点特征、OR surrogate score、dual/reduced-cost features、CDSTE validation proxy。
   - 动作：add node 或 swap-out/swap-in。
   - 奖励：CDSTE validation error reduction minus evaluation/search cost；可加入 potential-based reward shaping from OR surrogate。
   - 训练：先 behavior cloning 学 OR/local-search 轨迹，再 offline RL 或 conservative Q reranking。

5. **Certificates and interpretation**
   - Surrogate optimality gap。
   - Budget marginal value curve。
   - Layout stability across seeds/days。
   - Correlation between OR scores/dual prices and realized CDSTE error reduction。
   - Evaluation-call efficiency curve。

## Explicitly Rejected Complexity

- 不引入新 diffusion architecture，除非 CDSTE 原实现无法适配 fixed sensor mask。
- 不以 LLM/VLM 作为方法组件。
- 不优先做 mobile sensors，多阶段 relocation 留作后续论文。
- 不把 AMPL 只当 baseline solver；它必须产生 policy guidance 和证据。

## Novelty Positioning

相对 Xing et al. (2026)：他们是 TSLP-NFE 的 bi-level black-box + Embedding-BO；DOPPLER 是 OR-guided offline RL with certificates，目标是减少黑箱 estimator calls 并产生可解释部署建议。

相对 INSPIRE-GNN (2025)：他们是 DQN+GNN for bicycling volume estimation；DOPPLER 面向 motor traffic state imputation，复用 CDSTE，并将 MIP/LP dual information 作为核心机制。

相对 Owais (2024)：他们是 SAE + global sensitivity analysis for OD estimation；DOPPLER 是 CDSTE + mathematical programming guidance + sequential policy for network-wide traffic state estimation。

## Main Risks

1. OR surrogate 与真实 CDSTE objective 相关性不足。  
   缓解：保留 surrogate as candidate generation；用 RL reranking 学 residual。

2. CDSTE training cost 太高，无法支撑布局搜索。  
   缓解：先 frozen/pretrained estimator + short fine-tune；再只对 top layouts full training。

3. RL 不显著优于 local search。  
   缓解：把贡献收缩为 OR-guided amortized search；报告跨预算/跨网络 warm-start 优势。

4. TS 审稿人认为 DL 太黑箱。  
   缓解：主文放 formulation、bounds、budget curve、ablation 和 interpretability；CDSTE 放 estimator module。

## Success Criteria

- PeMS7_228 上预算 10%-30%，DOPPLER 相比最强非学习 baseline 至少降低 3%-5% MAE，或用少 50% estimator calls 达到同等误差。
- OR guidance deletion 会显著退化。
- 跨 PeMS7_228 -> PeMS7_1026 或 Seattle 的 zero/few-shot transfer 有正信号。
- 输出预算边际收益曲线，能支持“多装一个 sensor 的价值何时递减”的管理结论。
