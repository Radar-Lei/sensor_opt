<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| CLI flags -> evaluator | Untrusted experiment settings can request invalid rates, negative costs, or unsupported split/robustness modes. |
| Local dataset -> metrics artifacts | Raw local data stays ignored; generated artifacts must record configuration without exposing raw data. |
| Validation selection -> held-out test evidence | Perturbation semantics can accidentally leak held-out stress conditions into selection/tuning. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-05-01-01 | Tampering | robustness CLI flags | mitigate | Validate rates/scales are nonnegative and fractions are in [0,1]; fail fast with `ValueError`/`SystemExit` for unsupported modes. |
| T-05-01-02 | Information Disclosure | dataset boundary | mitigate | Tests use synthetic data; artifacts record config/metrics only and do not print raw dataset rows. |
| T-05-01-03 | Repudiation | metrics.csv provenance | mitigate | Write condition columns and config fields for every robustness run per D-12. |
