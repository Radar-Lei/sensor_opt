<interfaces>
Key evaluator contracts from the pattern map:
- solve_quadratic(observed_z, prior_z, sensors, matrix, obs_weight) returns (solution, lhs), where lhs is either a single N×N matrix for constant weights or a T×N×N stack for true time-varying weights.
- validation_mae(test, tod, distance, laplacian, precision, mean, std, sensors, args) returns the MAE for args.selection_method.
- evaluate_layout(...) remains the held-out evaluation path and must still compute all method rows plus certificates; validation_mae must not call it after this plan.
- Supported selection method names are historical_tod_mean, neighbor_average, gsp, and gls_map.
</interfaces>
